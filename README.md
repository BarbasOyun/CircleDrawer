# Circle Drawer

Simple Circle Drawer with egui in Rust
<br>

-Draw Circles with left click<br>
-Use sine & cosine to create the circle points<br>

Circles Options :<br>
-Radius (Change with mousewheel)<br>
-Segments<br>

Stroke Options :<br>
-Width<br>
-Color<br>
